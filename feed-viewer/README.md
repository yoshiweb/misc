# RSS Feed Viewer

このプロジェクトは、ユーザーが指定したRSSフィードの URL を読み込み、ページ上にフィードのタイトルと記事一覧を表示するシンプルなブラウザ向けツールです。

## 概要
- ユーザーが RSS フィードの URL を入力して追加できます。
- フィードの取得には無料のサードパーティ RSS→JSON API（例: rss2json）を利用しています。
- 取得したフィードはカード状のカラムに整形して表示し、各記事はリンクで開けます。
- 追加したフィードはローカルストレージに保存され、ページ再読み込み後も復元されます。

## 主な機能
- URL の追加（入力フォーム）
- ドラッグ＆ドロップは未対応（将来的な拡張候補）
- フィードごとの削除機能
- グリッドレイアウトでのレスポンシブな表示

## 使い方
1. ブラウザで [feed-viewer/index.html](feed-viewer/index.html) を開きます。
2. テキストボックスに RSS フィードの URL を入力して `Add RSS Feed` をクリックします。
3. フィードが追加され、各記事の一覧が表示されます。記事タイトルをクリックすると元のサイトを新しいタブで開きます。

## ファイル構成
- `index.html` — UI（入力欄、フィード表示領域）
- `app.js` — フィード取得・表示・ローカルストレージ管理・レイアウト調整のロジック
- `styles.css` — 表示用スタイル
- `README.md` — このファイル

## 技術メモ
- フィード取得: `https://api.rss2json.com/v1/api.json?rss_url=...` を使用
- レイアウト: CSS Grid を使い、各アイテムの高さに合わせて `grid-row-end` を動的に設定しています（`app.js` 内の `resizeGridItem` / `resizeAllGridItems` を参照）。

## 注意事項
- 外部の RSS→JSON サービスに依存しているため、そのサービスのレート制限や可用性に影響を受けます。
- プライバシー: フィード URL はブラウザのローカルストレージに保存されますが、フィードの取得自体は外部 API 経由で行われます。
