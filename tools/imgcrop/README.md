# 画像切り抜きツール

このツールはブラウザ上で複数画像をまとめてトリミングし、結果をZIPでダウンロードできるシンプルなフロントエンドアプリです。

## 主な特徴
- 画像はすべてブラウザ内で処理され、サーバーへ送信されません。
- Cropper.js を使って基準画像でトリミング範囲を指定できます。
- 他の画像には「支点（中央・左上・右上・左下・右下）」を基準に同じサイズで切り抜きます。
- トリミング結果をZIP（JSZip）で一括ダウンロード可能。
- CDN経由の TailwindCSS で簡易なUIを構築。

## 使い方
1. `index.html` をブラウザで開く（ローカルでファイルをダブルクリックでOK）。
2. ドラッグ＆ドロップ、または「画像を選択する」ボタンで複数画像を読み込む。
3. 左側のプレビューで1枚目（基準）に対してトリミング範囲を指定する。
4. 右側でアスペクト比や支点を設定する。
5. 「一括トリミングを実行」をクリックすると結果が表示される。
6. 「ZIPで一括ダウンロード」ボタンで全結果をまとめて取得する。

## サポート形式
- PNG, JPEG, GIF など一般的な画像フォーマット（ブラウザがサポートする形式）。

## 開発・カスタマイズ
- 依存ライブラリはすべて CDN を利用しているため、`index.html` をそのまま開くだけで動作します。
- 必要に応じて Cropper.js のオプションや UI 文言を変更してください。

## 注意事項
- JPEG など透過を持たない形式は背景を白で埋めて保存しています。
- 大きな画像を多数読み込むとブラウザのメモリを消費します。処理時間やメモリに注意してください。
