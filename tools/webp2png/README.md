# WebP → PNG Converter (webp2png)

このツールは、ブラウザ上で WebP 画像を受け取り、クライアントサイドで PNG に変換してプレビュー表示するシンプルなユーティリティです。サーバーへのアップロードは行わず、ローカルで完結します。

## 主な特徴
- ブラウザで完結するクライアントサイド変換
- ドラッグ＆ドロップまたはクリックでファイル選択
- 入力: WebP、出力: PNG（Canvas を利用した変換）
- 変換結果をその場でプレビュー表示

## 使い方

1. 画面中央のエリアに WebP ファイルをドラッグ＆ドロップするか、クリックしてファイルを選択します。
2. 変換が終わると下に PNG のプレビューが表示されます。必要であれば画像の右クリック→「画像を保存」で保存してください。

## 実装メモ
- `index.html` — UI（ドラッグエリア、入力、キャンバス、出力画像）
- `script.js` — ファイル入力、Canvas への描画、`toDataURL('image/png')` による PNG 変換の実装
- `styles.css` — 見た目のスタイル

変換処理の概要（`script.js`）:
- ドロップまたはファイル選択で File オブジェクトを取得
- `URL.createObjectURL` で Image に読み込み、`canvas` に描画
- `canvas.toDataURL('image/png')` で PNG データ URL を作成し、`img` 要素に設定

## 注意点
- ブラウザのセキュリティ設定やコーデック状況により一部の WebP が読み込めない場合があります。
- 大きな画像はメモリを多く消費します。必要に応じて描画時にリサイズ処理を追加してください。
