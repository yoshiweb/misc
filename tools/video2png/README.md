# 動画サムネイルジェネレーター

動画ファイルから指定間隔でサムネイル（画像）を生成し、まとめてZIPでダウンロードできるシンプルなブラウザツールです。

## 概要
- ローカルにある動画ファイルをブラウザ上で読み込み、指定した秒数間隔でフレームをキャプチャしてサムネイル画像を生成します。
- 生成したサムネイルは画面に一覧表示され、まとめてZIP（`thumbnails.zip`）としてダウンロードできます（JSZipを使用）。

## 使い方
1. `index.html` をブラウザで開く（おすすめは最新のChrome / Safari / Firefox）。
2. 「動画ファイルを選択」でローカルの動画ファイルを選ぶ。
3. キャプチャー間隔（秒）を指定する（デフォルトは `10` 秒）。
4. 「画像に変換」ボタンを押すと、指定間隔でサムネイルが生成され一覧に表示されます。
5. 「ZIPでダウンロード」を押すと、すべてのサムネイルが `thumbnails.zip` にまとめてダウンロードされます。

## 依存ライブラリ
- Tailwind CSS（CDN）— UI 用
- JSZip（CDN）— ZIP 生成用

これらは `index.html` 内で CDN 経由で読み込まれます。追加のインストールは不要です。

## 制限・注意点
- 大きな動画や長時間の動画はメモリ使用量や処理時間が増えるため注意してください。
- 一部の動画コーデックやブラウザ環境では `seek` が正確に動作しない場合があります。
- 動画が外部（リモート）にある場合、CORS 制約により読み込みやシークが失敗することがあります。ローカルファイルの利用を推奨します。
- 生成される画像はブラウザの canvas API を経由するため、画質・サイズはブラウザと動画の解像度に依存します。
