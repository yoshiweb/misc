# PNG/JPG to WebP Converter

このツールは、ブラウザ上で動作するシンプルな画像変換ユーティリティです。PNG または JPG/JPEG 画像を WebP 形式へクライアントサイドで変換し、その場でプレビューとダウンロードができます。

## 特徴
- ブラウザ内で完結（クライアントサイド変換、ファイルはサーバに送信されません）
- ドラッグ＆ドロップまたはファイル選択で簡単に変換
- 入力: PNG / JPG(JPEG)、出力: WebP
- 変換後のプレビュー表示とダウンロードリンクを提供

## 使い方
1. `index.html` をブラウザで開きます（ローカルファイルを直接開いて使えます）。
2. 画面中央のエリアへ PNG/JPG 画像をドラッグ＆ドロップするか、ファイル選択で画像を指定します。
3. 変換が完了するとプレビューが表示されます。ダウンロードリンクから WebP 画像を保存してください。

（内部的には `script.js` が Canvas を使って画像を読み込み、`toDataURL('image/webp')` で WebP に変換します。）

## 対応ブラウザ
WebP 形式のサポートが必要です。最新の Chrome、Firefox、Edge、Safari（比較的新しいバージョン）で動作します。古いブラウザでは WebP の生成または表示ができない場合があります。

## 開発・カスタマイズ
- 画質や圧縮率を調整したい場合は、`script.js` 内で `canvas.toDataURL('image/webp', quality)` の `quality` 引数を変更してください（0.0〜1.0）。
- 追加機能（複数同時変換、フォルダ処理、バッチダウンロードなど）は自由に実装できます。
