# PDF to Image Converter (pdf2img)

このツールは、ブラウザ上で PDF ファイルを読み込み、各ページを画像（PNG）として出力し、まとめて ZIP でダウンロードできるクライアントサイドのユーティリティです。

## 主な特徴
- ブラウザのみで動作（サーバへのアップロードは不要）
- PDF の各ページを高解像度の PNG に変換
- 変換した画像をページごとにプレビュー表示
- すべての画像を ZIP 化して一括ダウンロード

## 使い方
1. ドラッグ＆ドロップ、または「ファイルを選択」から PDF を指定します。
2. 変換が開始され、各ページの画像が下部に順次表示されます。
3. 変換が完了したら「画像をZIPでダウンロード」ボタンで一括取得します。

## 技術スタック
- PDF のレンダリング: `pdf.js`（CDN 経由で読み込み）
- ZIP 作成: `JSZip`（CDN 経由で読み込み）
- スタイル: Tailwind CSS（CDN）
- 実装ファイル: `index.html` 内のスクリプト（Canvas を使ってページをレンダリングし、`toDataURL('image/png')` で画像化）

## 注意点・制限
- 大きな PDF（ページ数や解像度が高いもの）はメモリを多く消費します。ブラウザや端末によっては処理に時間がかかったりクラッシュする可能性があります。
- 変換後の画像形式は PNG（可逆）です。必要に応じて JPEG や WebP に変換する処理を追加できます。
- ZIP 作成もクライアント側で行われるため、ファイルサイズやページ数によってはブラウザの制限に注意してください。

## カスタマイズ案
- 出力フォーマット（PNG/JPEG/WebP）の切替
- 画質やスケール（`scale` 値）の UI 制御
- バックグラウンド処理やワーカースレッド化による応答性改善
